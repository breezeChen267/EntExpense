# 台幣費用申請管理系統建置規畫書（V3.0）

## 一、系統架構設計（System Architecture）

本系統採用 **Clean Architecture + N-Layer 混合架構**，
達成業務邏輯與技術細節（資料庫、外部系統）解耦，並提升開發與維護效率。

---

### 1.1 邏輯分層與技術棧

| 層級       | 專案               | 技術                                                     |
| ---------- | ------------------ | -------------------------------------------------------- |
| 前端       | WFP.Web            | Angular 17+（Standalone、Signals）、Tailwind CSS         |
| API 層     | WFP.Api            | ASP.NET Core 8.0+、NeGotiate Authentication JWT、Swagger |
| 應用層     | WFP.Application    | Services、DTO、FluentValidation                          |
| 領域層     | WFP.Domain         | Entity（POCO）、Enum、Interface                          |
| 基礎建設層 | WFP.Infrastructure | EF Core、Serilog、AutoMapper、Dapper、IBM DB2 行整合     |

## 五、核心業務邏輯與防錯機制（Core Rules）

### 5.1 金額平衡檢核（Triple Check）

必須同時滿足：

- 受款平衡
  $\sum WFPPAYEA.APLAMT = WFPPAYE.APLAMT$

- 請款平衡
  $\sum WFPPAYEH.APLAMT = WFPPAYE.APLAMT$

- 分攤平衡
  $\sum WFPPAYED.ITEMAMT = 明細金額$

---

### 5.2 業務邏輯

- 受款人不可同時為廠商及員工

  - 受款人編號長度為 5 碼且存在於 WFPPERM 表示為在職員工
  - 受款人身份證字號存在於 WFPPERM 表示為在職員工

- 受款人新增後產生大寫金額

- 例如：總金額：NT$ 15,612 新台幣(大寫)壹萬伍仟陸佰壹拾貳元整

### 5.3 提供參考檔查詢指令

- 取得受款人

`SELECT CK0010,CK0040,CK0050,CK1120,CK1130 FROM dbo.GLPPAYU WHERE RTRIM(LTRIM(CK0010)) = '受款人編號'`

- 取得費用明細名稱及付款會計科目

`SELECT EXPTYPE, EXPID, ITEMDES, PAYACCT FROM dbo.GLPPAYI WHERE RTRIM(LTRIM(EXPID)) = '費用明細編號'`

- 取得請款費用(請款明細)清單

  - 說明 1：當 GLPPAYI.EXPID = 'A0001' THEN GLPPAYI.SHTCHK1 AND GLPPAYI.SHTCHK2 為必要檢附憑證編號，否則不允申請送出
  - 說明 2：GLPPAYI.SHTCHK1 ~ GLPPAYI.SHTCHK5 的值等於 GLPPAYS.SHTID
  - 說明 3： WFPPAYE.TOTSHEET 必須等於憑證筆數

`SELECT EXPID,ITEMDES,PAYACCT,ACPTER,WRTCODE,CASHCODE,SHTCHK1,SHTCHK2,SHTCHK3,SHTCHK4,SHTCHK5 FROM FINDB..GLPPAYI WHERE EXPTYPE = '請款種類編號' ORDER BY EXPID`

- 取得請款種類名稱

`SELECT EXPTYPE,ITEMDES FROM FINDB..GLPPAYT WHERE EXPTYPE = '請款種類編號'`

- 取得費用摘要清單

`SELECT ITEMNUM, ITEMDES FROM FINDB..GLPPAYX (NOLOCK) WHERE EXPID='費用明細編號'`

- 取得憑證清單

  `SELECT SHTID, SHTDESC FROM FINDB..GLPPAYS ORDER BY SHTID`

- 憑證編號=06 表示為發票。發票號碼已申請過不允再提申請以免重覆申請費用

`SELECT COUNT(\*) AS ReceiptCount FROM FINDB.dbo.WFPPAYES S LEFT JOIN FINDB.dbo.WFPPAYE E ON S.APLNO = E.PLNO WHERE S.SHTNO='發票號碼' AND E.APLSTAT NOT IN('2','3','9','D','T','W','F') AND S.SHTID='06'`

- 分攤方式清單

  - 舉例說明：WFPPERM.COMPCD 表示 WFPPERM 資料表的 COMPCD 欄位

  - IF WFPPERM.COMPCD='585' AND WFPPERM.DEPTCD = '330' THEN

    - IF WFPPERM.COMPCD IN ('5852','585C','585D') AND WFPPEM.JOBPOS = 'D03' THEN

    `SELECT SPRID, ITEMDES FROM FINDB..GLPPAYR R WHERE R.SPRID <> '0000' ORDER BY SPRID`

    - ELSE

    `SELECT SPRID, ITEMDES FROM FINDB..GLPPAYR R WHERE SPRID <= '0004' AND SPRID <> '0000'  ORDER BY SPRID`

    - ELSEIF WFPPERM.BRHCOD='5850' AND WFPPERM.DEPTCD IN ('400','800') AND NOT(WFPPERM.BRHCOD='5850' AND WFPPERM.DEPTCD = '106')

    `1SELECT SPRID, ITEMDES FROM FINDB..GLPPAYR R   WHERE SPRID <= '0004' ORDER BY SPRID`

    - END IF

- 分攤單位

  - IF EXPTYPE IN ('F','H') OR (EXPTYPE = 'C' AND EXPID = 'C0009') THEN

  `SELECT FINDB..GLPDEPT.COMPID, FINDB..GLPCOMP.COMPBNM AS COMPNAM, FINDB..GLPDEPT.DEPTNUM, '' AS DEPTNAM  FROM FINDB..GLPDEPT  LEFT OUTER JOIN FINDB..GLPCOMP ON FINDB..GLPDEPT.COMPID=FINDB..GLPCOMP.COMPID`

  - IF EXPTYPE = 'C' AND EXPID >= 'C0009' THEN

  `SELECT FINDB..GLPDEPT.COMPID, FINDB..GLPCOMP.COMPBNM AS COMPNAM, FINDB..GLPDEPT.DEPTNUM, '' AS DEPTNAM FROM FINDB..GLPDEPT  LEFT OUTER JOIN FINDB..GLPCOMP ON FINDB..GLPDEPT.COMPID=FINDB..GLPCOMP.COMPID WHERE   FINDB..GLPDEPT.COMPID='5850' AND FINDB..GLPDEPT.DEPTNUM='001' AND RIGHT(RTRIM(FINDB..GLPDEPT.DEPTNAM), 1) = ';' ORDER BY FINDB..GLPDEPT.COMPID, FINDB..GLPDEPT.DEPTNUM`

  - ELSE

  `SELECT FINDB..GLPDEPT.COMPID, FINDB..GLPCOMP.COMPBNM AS COMPNAM, FINDB..GLPDEPT.DEPTNUM, '' AS DEPTNAM  FROM FINDB..GLPDEPT  LEFT OUTER JOIN FINDB..GLPCOMP ON FINDB..GLPDEPT.COMPID=FINDB..GLPCOMP.COMPID WHERE FINDB..GLPDEPT.COMPID LIKE '585%' AND FINDB..GLPDEPT.DEPTNUM='001' AND RIGHT(RTRIM(FINDB..GLPDEPT.DEPTNAM), 1) = ';' AND NOT(RIGHT(RTRIM(FINDB..GLPCOMP.COMPNAM), 1) = '*') ORDER BY FINDB..GLPDEPT.COMPID, FINDB..GLPDEPT.DEPTNUM`

  - END IF

  - ELSE

`SELECT FINDB..GLPDEPT.COMPID, SUBSTRING(FINDB..GLPCOMP.COMPBNM,8,5) AS COMPNAM, FINDB..GLPDEPT.DEPTNUM, FINDB..GLPDEPT.DEPTNAM FROM FINDB..GLPDEPT  LEFT OUTER JOIN FINDB..GLPCOMP ON FINDB..GLPDEPT.COMPID=FINDB..GLPCOMP.COMPID WHERE FINDB..GLPDEPT.COMPID LIKE '585%' AND NOT (RIGHT(RTRIM(FINDB..GLPDEPT.DEPTNAM), 1) IN (';', '*'))   ORDER BY FINDB..GLPDEPT.COMPID, FINDB..GLPDEPT.DEPTNUM`

- END IF

- 匯出銀行
  - payacct = GLPPAYI.PAYACCT
    `SELECT CCY,FLGRPT, AC AS ACCOUNT,FISCODE AS ACCTBNK,ACNAME AS ACCTFULLNAME FROM FINDB.CPDLIB.CPPAC WHERE GLACCT='" & payacct & "' AND COID='5850'  ORDER BY ACNAME COLLATE Chinese_Taiwan_Stroke_BIN2`
